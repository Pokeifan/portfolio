Mes Projets

Jeu de Bataille

Ce projet implémente un jeu de cartes classique où deux joueurs s'affrontent pour voir qui possède la carte la plus forte. Le jeu gère les tours, les égalités et les batailles entre les joueurs. Une fois le jeu terminé, un gagnant est annoncé en fonction des cartes remportées.

Langage : Python | Framework : Aucun

Manga/Anime Finder

Ce projet est une SPA qui permet aux utilisateurs de rechercher des mangas et animés par nom, en récupérant des informations détaillées telles que l'image de couverture, la description, le genre, et l'année de sortie grâce à l'API Jikan. L'application affiche les résultats dynamiquement sans rafraîchir la page, offrant une expérience utilisateur fluide et rapide.

Langages : HTML, CSS, JavaScript | API : Jikan
